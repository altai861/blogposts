# Хэрхэн сайжрах вэ?

    Ойрын үед гэлтгүй ер нь л надад их ажиглагдаж байгаа зүйл бол өөрийгөө хангалттай дайчилж өөрийн нөөц бололцоог шахаагүй
байж элдэв дур хүсэлдээ автаж өөрийгөө хэтэрхий их шагнаж байна.
    Сүүлд миний нэг бодсон зүйл бол амархан санагдахаа болих үеэс эхлэн жинхэнэ ажил эхэлдэг. Юмаа хийх үедээ юмаа л хий.
Тэр зүйлээсээ таашаал ав. Ээж өнөөдөр 150'000 төгрөг явуулсан байна. Чи ичэх хэрэгтэй. Нас биенд хүрсэн эрэгтэй хүн байж ээжээсээ
мөнгө авч байгаадаа санаа зовох хэрэгтэй. Хөдөлмөрлө. Хийх ёстой юмаа хий. Сайн унт. Сайн ид. Бас өөрийгөө шагнамаар байвал зоос шид.
